# Snake Game

A classic Snake game served via a local Python HTTP server.

## Requirements

- Python 3.x (no extra packages needed)

## How to Start

1. Open a terminal and navigate to this directory:

   ```bash
   cd path/to/snake_game
   ```

2. Start the server:

   ```bash
   python server.py
   ```

3. Open your browser and go to:

   ```
   http://localhost:8080
   ```

4. Press **Start Game** and play!

## Controls

| Key | Action |
|-----|--------|
| Arrow Keys / WASD | Move the snake |
| P | Pause / Resume |

## Stopping the Server

Press `Ctrl+C` in the terminal to stop the server.
